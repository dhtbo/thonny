======
Thonny
======

Thonny 是一个 Python 编程 `IDE <https://en.wikipedia.org/wiki/Integrated_development_environment>`_。 

安装环境

1. 安装python
2. 安装必备库
  pip install -r requirements.txt
3. 运行
  # 直接运行源码

  python -m thonny

  # 本地安装thonny

  python setup.py install
  thonny
